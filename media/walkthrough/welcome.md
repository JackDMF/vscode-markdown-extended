## Markdown Extended Pro

Supercharge VS Code's built-in Markdown with extra syntax in the live preview and a what-you-see-is-what-you-get exporter.

**What you get**

- ✍️ **Rich syntax** — sidenotes, marginal notes, sidebars, admonitions, containers, footnotes, definition lists, marks, sub/superscript, keyboard keys, abbreviations, inline attributes, and a table of contents.
- 📤 **Self-contained export** to HTML, PDF, PNG, and JPEG — styles and images embedded, matching the preview.
- 🧜 **Mermaid diagrams** are rendered to inline SVG in exports.
- ✏️ **Editing helpers** — formatting toggles, table tools, and paste-as-table.
- 🎨 **Two editor color themes** for extended-syntax highlighting.

Open the Markdown **preview** beside your file (`Ctrl/Cmd+K V`), then work through the steps on the left. Each one shows a snippet you can drop into a `.md` file and watch render live.
